# Plex 有声书播放器

一款为重视舒适性与简洁性的 Plex 用户打造的私有有声书播放器。本应用将在很长一段时间内保持免费。我希望收集一些反馈，并在未来开发更多实用的应用。如果你觉得这款应用很棒，欢迎在 App Store 给我五星好评，并关注我的 GitHub 账号。感谢你的使用。

**你可以在 Issues 标签页中提交 Bug 反馈。**

本应用可直接连接到你自己的 Plex 服务器，并提供专为有声书优化的聆听体验。

---

## 功能特性

- ⏭️ 跳过片头和片尾，畅听不中断
- ⏰ 睡眠定时器，安心入睡
- 📡 支持 AirPlay
- 📚 按章节浏览与跳转
- ✍️ 按作者浏览有声书
- 🎨 可自定义主题颜色
- 🌍 多语言界面支持

---

## 支持的语言

- 英语  
- 西班牙语  
- 日语  
- 中文  

---

## 为舒适聆听而设计

这不是一款通用的媒体播放器。

它是一款**专注、私有且舒适的有声书应用**，旨在让长时间聆听变得轻松自然——无论是在通勤途中、放松时，还是入睡前。

---

## 隐私

- 不进行用户追踪
- 不使用分析工具
- 不收集任何数据
- 不使用开发者自建服务器

所有数据仅保存在用户的本地设备中。
