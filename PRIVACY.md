# Privacy Policy

This app is developed and maintained by an independent developer.

## No Data Collection
The developer does **not collect, track, analyze, or monitor any user behavior** in any form.

The app does **not collect, store, or transmit any personal data**, usage data, analytics data, or identifiers.

## Local-Only Data Storage
All data used by the app, including but not limited to:
- Plex authentication information
- Server connection details
- Playback progress
- User preferences

is stored **locally on the user’s device only**.

The developer does **not** have access to this data.

## No Developer Servers
This app does **not** use, operate, or connect to any servers owned or controlled by the developer.

There is **no backend service**, **no cloud storage**, and **no remote logging** involved in the operation of this app.

## Plex Integration
The app connects directly to the user’s own Plex server and official Plex services solely for authentication and media access.

All communication with Plex services occurs directly between the app and Plex.  
The developer does not intercept, proxy, store, or analyze any Plex-related data.

## Third-Party Sharing
The developer does **not share any data with third parties**.

Any data exchanged with Plex services is strictly required for core app functionality and is handled according to Plex’s own privacy policy.

## Changes to This Policy
If the app’s behavior regarding data or privacy changes in the future, this privacy policy will be updated accordingly.
