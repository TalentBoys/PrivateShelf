# Plex オーディオブックプレーヤー

快適さとシンプルさを重視する Plex ユーザーのために設計された、プライベートなオーディオブックプレーヤーです。このアプリは長期間にわたって無料で提供される予定です。フィードバックを集め、今後さらに便利なアプリを開発したいと考えています。もしこのアプリが気に入ったら、App Store で ★★★★★ の評価を付け、私の GitHub アカウントをフォローしてください。ご利用ありがとうございます。

**不具合の報告は Issues タブから行えます。**

このアプリはご自身の Plex サーバーに直接接続し、オーディオブックに最適化されたリスニング体験を提供します。

---

## 機能

- ⏭️ イントロやアウトロをスキップして途切れない再生
- ⏰ 就寝前も安心なスリープタイマー
- 📡 AirPlay 対応
- 📚 チャプターごとの閲覧と操作
- ✍️ 著者別でオーディオブックを閲覧
- 🎨 テーマカラーのカスタマイズ
- 🌍 多言語インターフェース

---

## 対応言語

- 英語  
- スペイン語  
- 日本語  
- 中国語  

---

## 快適なリスニングのために設計

これは汎用的なメディアプレーヤーではありません。

通勤中、リラックス中、眠りにつく前など、長時間のリスニングを自然で快適にするために作られた、**集中できるプライベートなオーディオブックアプリ**です。

---

## プライバシー

- ユーザー追跡なし
- アナリティクスなし
- データ収集なし
- 開発者が運営するサーバーなし

すべてのデータはユーザーのデバイス内にのみ保存されます。
